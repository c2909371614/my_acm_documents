WC的困难贪心

# dec
给出n个a[i],b[i], 要求选出不多于A个a[i],不多于B个b[i],(选择a[i]就不能选择b[i]),使得两者总和的和最大

#input
一个T，表示有T组数据
三个数n,A,B
n个数，表示a[i]
n个数，表示b[i]

1 <= A,B <= n <= 20
1 <= a[i],b[i] <= 1e5

1 <= A,B <= n <= 1e5
1 <= a[i],b[i] <= 1e5



#output
一个数，表示所求的最大值

#sample input
1
5 3 3
1 2 3 4 5
5 4 3 2 1

#sample output
21
